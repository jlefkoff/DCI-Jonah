Timeline

How this project came about

  • July 2018

    Network install

    I carefully planned and installed the server rack and access points all around my house. Then I spent a day configuring all of the equipment.

  • August

    Satisfied with the network!

    A month after the install, everyone was very satisfied with how the network turned out.

  • September

    Brainstorming my DCI project

    I thought a lot about what I shoud do for my first DCI project, and I decided to take the knowledge I had gained from building my own home network and apply it to a website.

  • October to Now

    Designing and coding

    I learned a lot about how SVG images work and how I could implement them into my own code. I built the map in illustrator and then I edited the code to make each icon have its own name. This whole process was really fun!

The Server Rack

I chose to use a Ubiquiti networks router and switch. I also have a power supply and a Comcast modem. Throughout the install, I had to clean up and organize many cables.

A wireless access point

I installed 6 wireless access points around the house for the best possible wireless coverage.

A wired connection

I ran ethernet cables throughout the house to computers and stationary devices, desktops and printers, to get a better connection for those devices.
